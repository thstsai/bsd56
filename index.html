📋 競賽說明

本培訓平台針對參加第56屆全國技能競賽 商務軟體設計（科目代碼：09）的選手設計，課程架構融合 55分區、55全國、56分區 三套試題的重點技術。

📁 SA&DB 模組 — SQL Server 2022, 3NF正規化, ER圖, Use Case圖
🖥 Software 模組 — C# .NET 8, WinForms, ADO.NET
📱 App 模組 — Android Studio Ladybug, Kotlin 2.1.10, Retrofit
